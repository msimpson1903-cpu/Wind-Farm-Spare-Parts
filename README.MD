# Wind Farm Spare Parts Inventory

This project is a spare parts inventory system for wind farm operations.

Goals:
- Track stock levels
- Manage suppliers
- Log stock movements
- Track turbine spare parts
- Generate low stock alerts

System Structure:

Tables:
- Parts Master
- Stock Movements
- Suppliers
- Locations
- Purchase Orders

Future Features:
- QR code scanning
- Technician issue forms
- Automatic reorder alerts
